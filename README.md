## Описание проекта

Пет проект реализует калькулятор.

Стек: HTML, CSS, JS, Webpack

## Установка и запуск

Для установки и запуска проекта необходимо выполнить команды

```
npm install
npm run start
```

## Сборка

```
npm run build
```
